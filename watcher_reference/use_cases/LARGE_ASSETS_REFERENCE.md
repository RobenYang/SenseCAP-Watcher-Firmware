# Large Assets Reference

This private repo keeps the official `watcher_reference/use_cases/` tree as a lightweight reference only.

To reduce repo size, large generated UI assets, bundled firmware binaries, and model files were removed from this private branch.

Authoritative source for all removed files:

- Official repository: `https://github.com/Seeed-Studio/SenseCAP-Watcher-Firmware`
- Recommended branch/tag to search first: `main`
