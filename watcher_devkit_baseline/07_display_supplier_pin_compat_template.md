# 07 Display Supplier Pin Compatibility Template

## 目标
用于向屏厂确认“是否可原生直插 SenseCAP Watcher 主板 LCD 座（J1）”。

本模板只做硬件插口与引脚兼容确认，不包含固件适配。

## 设备基准（Watcher 主板）
- 来源：`OSHW-SenseCAP-Watcher` 原理图 `SenseCAP_Watcher_v1.0_SCH.pdf`，`LCD&TP` 页（J1）。
- 连接器：`J1`，`28 pin`。
- 逻辑电平：`3.3V`。
- 显示总线：`QSPI (SPI3)`。
- 触摸总线：`I2C1 + TP_INT`。

## 一票否决项（任一不满足即非原生直插）
1. 模组 FPC 引脚数不是 `28 pin`（且无同接口重定义版本）。
2. 供电轨不兼容（例如要求 `1.8V` 逻辑且无板上电平转换）。
3. 显示数据接口不是 `QSPI`（仅 MIPI DSI 且不支持 QSPI）。
4. 触摸接口不是 `I2C + INT`（且无法改 pin 定义）。
5. 引脚定义无法按下表 pin-to-pin 对齐。

## 自定义固件模式下可简化项
你不使用官方固件时，本模板可以按目标分两档使用：

### A. 目标是“原生直插 Watcher 主板 J1”
- 不能简化：`J1 28pin` 逐 pin 对版表、供电轨、电平、接口类型、机械尺寸。
- 可简化：不要求供应商提供“官方驱动适配说明”，只要提供基础初始化建议即可。

### B. 目标是“允许转接板/重排 FPC，不追求 pin-to-pin 直插”
- 可以简化：删除 `J1 逐 pin 对版表`，改为“功能信号组匹配”。
- 必留项：
  1. 供电需求（VCI/VDDIO/VBAT/VSP/VSN 等）与上电时序图
  2. 显示接口（QSPI 或 MIPI）与时钟上限
  3. 触摸接口（I2C 地址、INT/RST、工作电压）
  4. 机械（外径 48mm、厚度、FPC 出线方向）

## 机械确认项（按你的目标外形）
- 目标外径：`48mm`（允许公差请供应商明确）。
- 触摸盖板外径 / 可视区 / AA（Active Area）尺寸。
- FPC 出线方向（相对 12 点钟方向的角度）。
- FPC 厚度、补强板尺寸、插入方向（上接/下接）。
- 屏体总厚度（含 TP）。

## 你可以直接发给供应商的话术
> 我们需要一款可直接替换 SenseCAP Watcher 的圆形显示模组。请按附件表格逐 pin 回填你们 FPC 与我方 J1(28pin) 的对应关系，并确认是否无需转接板即可直插。若不能直插，请给出可定制 FPC 重排线方案（最小起订量、打样周期、NRE 费用）。

## 回收资料清单（要求供应商提供）
1. 屏幕规格书（含 pin 定义、供电时序、初始化建议）。
2. FPC 尺寸图（含补强与开口位置）。
3. 触摸 IC 型号与 I2C 地址。
4. 若需定制：pin 重排后的确认图（PDF + 可编辑源文件）。
